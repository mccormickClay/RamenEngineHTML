◆ LoadUserResources()

static void AssetManager::LoadUserResources ( )
staticprivate

Loads User resources. This is found in the user folder and DOES NOT get called from a GameObject. Load all Assets (Models, Textures, Shaders, Images) here.

Example:

void AssetManager::LoadUserResources() { LoadModel("t99turret", "t99turret.azul"); LoadTexture("t99track", "track.tga"); }

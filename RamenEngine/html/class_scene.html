Detailed Description

The entity that holds all GameObjects at a given time. It controls whether Objects are Drawn, Updated, Can use alarms, send an input, and collide with other objects. Commands are needed to "Submit" registration requests to the Scene. multiple scenes can be used and switched between one another.

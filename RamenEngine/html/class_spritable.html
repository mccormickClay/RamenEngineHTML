Detailed Description

Allows objects to draw sprites over 3D objects.

All Commands must be Registered and Deregistered by the GameObject to be used correctly. All GameObjects have Submit2DDrawRegistration and Submit2DDrawDeregistration.

Objects and Sprites rendering in Draw2D will always be drawn infront of objects rendering in the original Draw.

Sprites can be either Controlled by the user or By the inhouse SpriteFactory.

Clay, 3/7/2018.

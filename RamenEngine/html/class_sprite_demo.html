Detailed Description

In this Demo, sphere is used to test how Alarmable workds and how to make a constant loop of alarms. The Alarms are done within the GameObject (because they have the Alarmable Component).

This Demo will show you how to register Alarms and how to set them correctly.

◆ Initialize()

void SpriteDemo::Initialize ( )
virtual

This is a virtual for user Scenes. This allows Initialize to be used in any Scene and calls back to this function first. This allows Users to create "new" Objects inside their scene.

Initialize MUST be used inside the user's scene to create GameObjects correctly.

UserScene::Initialize() { pAirplane = new Airplane(); }

◆ SceneEnd()

void SpriteDemo::SceneEnd ( )
virtual

This is a virtual function that is needed for the User Scene to allow GameObjects to be properly deleted. Before a scene is deleted, this function is called allowing the user to delete their GameObjects and Factories correctly.

To avoid issues, add SceneEnd to your Scenes functions.

void TestScene::SceneEnd() { delete pAirplane; delete gOFrigate; EnemyTankFactory::Terminate(); BulletFactory::Terminate(); }

class  GameObject
 The game entity that holds most information to see, control, and interact with. A GameObject is made up of multiple Components: Drawable, Updatable, Alarmable, Inputable, and Collidable. GameObjects are used within a Scene and can use any of these Components (depending if the user asks for submission).

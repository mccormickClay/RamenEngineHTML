Detailed Description

A camera tool for the user. Allows the User to ask for Cameras in use or to set them to a new Camera given by the User. This is use for Rendering a GameObject into the Scene.

void SphereObject::Draw() { GraphicObj->Render(CameraTool::GetCurrentCamera()); }

Detailed Description

Allows Gameobjects to be updated each frame.

All Commands must be Registered and Deregistered by the GameObject to be used correctly. All GameObjects have SubmitUpdateRegistration() and SubmitUpdateDeregistration();

◆ Update()

void Updatable::Update ( )
privatevirtual

Callback for all GameObjects to be updated by the Scene. Update will likely be used with GraphicsObject SetWorld(Matrix& world).

Example

void Bullet::Update()
{
Matrix world;
Vect Trans;
Trans += Vect(0, 0, 1) * Rot * BulletSpeed;
world = Scale* Rot * Matrix(TRANS, Trans);
GraphicsObj->SetWorld(world);
}

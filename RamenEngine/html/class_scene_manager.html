◆ StartScene()

static Scene* SceneManager::StartScene ( )
staticprivate

This function is specifically used to have the user start the game with a specific scene. This is found in the USER folder called: UserStartScreen.cpp.

The User needs to include the scene.h that will be used. and have a new scene of that object be returned in the function.

Example:

Scene* SceneManager::StartScene() { return(new TestScene); }

Clay, 3/10/2018.

Returns
Returns the first scene to be used in the game. Not having a scene here will cause a crash.

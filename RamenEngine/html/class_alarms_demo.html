Detailed Description

In this Demo, sphere is used to test how Alarmable workds and how to make a constant loop of alarms. The Alarms are done within the GameObject (because they have the Alarmable Component).

This Demo will show you how to register Alarms and how to set them correctly.

Clay, 3/7/2018.

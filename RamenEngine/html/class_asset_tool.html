Detailed Description

An user asset tool that is automatically used by GameObjects. AssetTool Gets information based on the asset type and the key the user needs.

GraphicObj = new GraphicsObject_WireframeConstantColor(AssetTool::Model("radModelName"), AssetTool::Shader("randomShader"), currentColor);

Clay, 3/9/2018.

Detailed Description

Allows Gameobjects to be drawn on screen.

All Commands must be Registered and Deregistered by the GameObject to be used correctly. All GameObjects have SubmitDrawRegistration() and SubmitDrawDeregistration();

◆ Draw()

void Drawable::Draw ( )
privatevirtual

Callback for all GameObjects to be drawn on screen. Draw will likely be used with GraphicsObject Render(Camera* pCam).

Example

void Bullet::Draw()
{
GraphicsObj->Render(CameraTool::GetCurrentCamera());
}

Reimplemented in SphereObject, SpriteDemoObject, Axis_Default, and SpriteDemo3DObject.
